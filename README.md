# java-shareit

Онлайн платформа для аренды(шеринга) вещей.

### Стек
- Java 11
- Spring Boot
- Hibernate
- Lombok
- RestTemplate
- MockMVC
- Maven
- Junit
- PostgreSQL
- Docker

## Приложение состоит из двух микросервисов:
### gateway:
Получение и валидация запросов пользователя.

### server:
Сервис, отвечающий за основной функционал приложения:
  - Создание новых пользователей, обновление данных, удаление;
  - Создание, удаоение, редактирование и бронирование вещей;
  - Поиск вещей по тексту, с возможностью фильтрации и постраничной выдачей;
  - Подтверждение или отклонение заявок на аренду владельцем;
  - Создание запросов на вещи, которые еще не представлены в приложении;
  - Получение списка всех бронирований пользователя, с возможностью сортировки и фильтрации;
  - Получение списка всех бронирований вещей, принадлежащих пользователю, с возможностью сортировки и фильтрации;
  - Получение списка всех вещей пользователя;
  - Публикация отзывов на вещи;
  - Получение пользователем всех его запросов, с возмодностью фильтрации и сортировки;
  
## Запуск приложения:
1) Сборка - mvn clean package
2) Запуск - docker-compose up
